1. Exact Solution
2. Euler Method
3. Modified Euler Method (Heun's Method)
4. Adams-Bashforth Predictor and Adams-Moulton Corrector
5. Taylor Series Method (2nd Order)
6. Runge-Kutta 4th Order Method
7. Runge-Kutta 2nd Order (RK2)
8. Milne's Method
